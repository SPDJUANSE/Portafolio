Sobre mí

Soy un apasionado desarrollador de software con enfoque en soluciones web y móviles, actualmente cursando el 8° semestre de Ingeniería de Software en la Universidad Manuela Beltrán.

Con experiencia en desarrollo tanto frontend como backend, he trabajado con tecnologías como Java, Spring Boot, JavaScript, React y Python. Además, poseo

conocimientos sólidos en PostgreSQL, APIs REST, Docker, Git y metodologías ágiles como SCRUM.
